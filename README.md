# 🃏 Card Logic Master (トランプ・ロジック・マスター)

二分探索アルゴリズムと論理的推論を駆使して、隠されたトランプを当てるブラウザゲームです。


## 🎮 ゲームモード
プレイヤーのレベルに合わせて3つの難易度を用意しています。

| モード | 対象 | 質問回数 | 特徴 |
| :--- | :--- | :--- | :--- |
| **ノーマル** | 数字のみ (1枚) | 4回 | 絞り込みアルゴリズムの基本を学べます。 |
| **ハード** | マーク + 数字 (1枚) | 5回 | 効率的な質問順序が鍵となります。 |
| **ヘル** | マーク + 数字 (2枚) | 10回 | **自動消去なし。** ログから2枚の状態を推論する最難関モード。 |

## 🛠 技術スタック
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
- **Deployment:** GitHub Pages

## 🧠 ロジックの解説
このゲームは、効率的に情報を絞り込む「二分探索」の考え方をベースに設計されています。
- 13枚の数字を特定するのに必要な最小質問回数は $\log_2 13 \approx 3.7$ 回。
- 52枚（ハードモード）を特定するのに必要な回数は $\log_2 52 \approx 5.7$ 回。

ヘルモードでは、2つの独立した変数を同時に追いかける必要があるため、非常に高い論理的思考力が要求されます。
